14 ## @class CorrelationVector
15 # @brief CorrelationVector estimates the correlation vector and delay between
16 # a signal and a time-delayed measurement of that signal
17 #
18 # @details
19 # This class contains an estimator which estimates the correlation vector
20 # between a signal (the #__trueSignal__) and measurements of that signal. This
21 # correlation vector is then used to estimate the delay between the
22 # #__trueSignal__ and the measurements of that signal.
23 #
24 # The estimator in this class currently assumes that the signal source is
25 # "distant," or infinitely far away. This implies that the unit vector to the
26 # signal source is perfectly known, and not changing. A later implementation
27 # could include the option of having a non-distant signal source, in which the
28 # unit vector is changing as a function of position and therefore uncertain.
29 #
30 # @note This class is essentially an implementation of the estimator presented in
31 # <a href="https://doi.org/10.2514/1.G002650">
32 # Recursive Range Estimation Using Astrophysical Signals of Opportunity</a>,
33 # J. Runnels, D. Gebre, Journal of Guidance, Control and Dynamics, 2017. Some
34 # equations from the paper are included in the class documentation for
35 # reference. A more detailed discussion and derivation of the estimator can
36 # be found in the journal article..

768  ## @fun #computeSignalTDOA computes the delay between the #__trueSignal__ and
769  # measurements based on a correlation vector
770  #
771  # @details The #computeSignalDelay function is a rudimentary function
772  # which takes an estimate of the correlation vector and uses it to
773  # estimate the location of the peak. It functions by finding the tap with
774  # the maximum value, and then fitting a quadratic to the points
775  # surrounding the maximum value tap. The number of points to which the
776  # quadratic is fitted is determined by the value of #peakFitPoints; the
777  # number of points is equal to \f$2 * n + 1\f$ where \f$n = \f$
778  # #peakFitPoints.
